Allows overriding the Slack Plugin Proxy Server specified in the global configuration.
Acceptable formats are: proxy-server.example.com | proxy-server.example.com:9323.
If no port is specified this will default to use port 8080.
slackSend channel: "#channel-name", proxyServerUri: "proxy.server.com:4382", message: "Build Started: ${env.JOB_NAME} ${env.BUILD_NUMBER}"
